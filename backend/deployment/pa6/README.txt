Explain the concept of clustering in your own words.
Clustering involves connecting multiple servers to work together as a single system, with each server assigned a specific role, like running application code, managing databases, or storing files. This approach optimizes performance and reliability by allowing each server to focus on its dedicated task while communicating with others.

Explain the concept of load balancing in your own words.
Load balancing is distributing incoming traffic evenly across multiple servers to prevent overload and ensure efficient operation.
